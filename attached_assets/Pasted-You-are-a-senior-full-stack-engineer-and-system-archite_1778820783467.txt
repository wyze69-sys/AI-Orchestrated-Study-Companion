You are a senior full-stack engineer and system architect.

Your task is to build a production-style MVP called:

AI-Orchestrated Study Companion

This is a full-stack split-screen web application where students upload study materials and interact with an AI tutor grounded ONLY in their uploaded documents.

==================================================
TECH STACK
==================================================

Frontend:
- React.js
- React Router
- Tailwind CSS
- Axios
- react-markdown
- remark-gfm

Backend:
- Node.js
- Express.js
- mysql2
- JWT authentication
- bcrypt
- Multer
- Server-Sent Events (SSE)

Database:
- MySQL

AI:
- Google Gemini API
- Ollama support via adapter pattern

==================================================
CORE FEATURES
==================================================

Users can:
- Register and login securely
- Create study sessions
- Upload study documents
- Chat with AI using uploaded docs as context
- Receive streamed AI responses
- Reopen old sessions with history preserved
- Use quick AI actions like summarizing and quiz generation
